                                                          تقویم هجری شمسی

                                                          این کتابخانه به صورت پیشفرض از تقویم میلادی و زبان انگلیسی استفاده میکنه و direction پیشفرضش هم چپ به راست در نظر گرفته شده. به همین دلیل در صفحاتی که direction آن راست به چپ هست، باید مقدار calendarPosition را برابر با auto-right قرار بدید.

                                                          فرمت پیشفرض نمایش تاریخ در input به صورت YYYY/MM/DD تعریف شده. در صورت نیاز و در صفحات راست به چپ، میتونید مقدار format رو به DD/MM/YYYY تفییر بدید.

                                                          import React from "react"
import DatePicker from "react-multi-date-picker"

export default function example() {
  return (
    <DatePicker
      calendar="persian"
      locale="fa"
      calendarPosition="auto-right"
      format="DD/MM/YYYY"
    />
  )
}

                                                          مشخص کردن زمان دلخواه

                                                          همانطور که در مثال های بالا مشاهده کردید هیچ مقدار اولیه ای به عنوان تاریخ برای دیتپیکر مشخص نشده، بنابراین با اولین کلیک، مقدار تاریخ آن برابر با همان روز در نظر گرفته میشه.

                                                          در صورت مشخص نکردن مقدار اولیه، تاریخی که توسط کاربر انتخاب میشه در یک استیت داخلی نگداری میشه، بنابراین با رندر شدن دوباره توسط ری اکت ، آن مقدار از دست نمیره.

                                                          ولی در صورتی که تاریخ بخصوصی رو به عنوان مقدار اولیه در نظر دارید، میبایست آن را در یک استیت نگداری کنید و زمانی که کاربر تاریخ جدیدی را انتخاب میکنه ،توسط متد onChange آن را در استیت خود بروزرسانی کنید تا با رندر شدن دوباره، تاریخ بروزرسانی شده تون از دست نره.

                                                          مقدار اولیه تاریخ میتونه از جنس Date , DateObject , Number و یا String باشه ولی به محض انتخاب تاریخ جدید توسط کاربر ، Type آن به DateObject تغییر میکنه.

                                                          بنابراین اگر در هنگام سابمیت فرم خود میخواید تاریخ انتخاب شده ی کاربر رو به Date جاوااسکریپت تبدیل کنید ، میتونید خیلی راحت از متد toDate() استفاده کنید.

                                                          برعکس مورد بالا، برای تبدیل String یا Date به DateObject باید آن مقدار را در کانستراکتور DateObject قرار بدید.ولی با توجه به اینکه تقویم پیشفرض DateObject، میلادی هست برای تبدیل String تاریخ شمسی به DateObject به جای خود String، یک آبجکت همانند مثال پایین در کانستراکتور قرار میگیره.

                                                          در این مثال، مقدار اولیه تاریخ از جنس String و زمان آن اول فروردین 1400 در نظر گرفته شده و توسط دکمه سابمیت تاریخ انتخاب شده کاربر از حالت DateObject به Date جاوا اسکریپت تبدیل شده و در یک span نمایش داده میشه.

                                                          import React, { useState } from "React"
import DatePicker, { DateObject } from "react-multi-date-picker"

export default function example() {
  const [value, setValue] = useState("1400/01/01")
  const [date, setDate] = useState()

  return(
    <div>
      <DatePicker
        calendar="persian"
        locale="fa"
        calendarPosition="auto-right"
        value={value}
        onChange={setValue}
      />
      <button
        onClick={() => {
          if (typeof value === "string") return setDate(
            //هنوز تاریخ جدبد توسط کاربر انتخاب نشده
            //و مقدار آن 1400/01/01
            //میباشد. بنابراین اول آن را به صورت دستی به
            //DateObject
            //تبدیل کرده و سپس تاریخ تولید شده را به
            //Date 
            //جاوا اسکریپت تبدیل میکنیم
            new DateObject({ 
              calendar: "persian", 
              date 
            }).toDate()
          )
          //تاریخ جدید توسط کاربر انتخاب شده و با متد
          //toDate()
          //قابل تبدیل میباشد
          setDate(value.toDate())
        }}
      >
        سابمیت
      </button>
      <span>{typeof date === "undefined" ? "" : ("" + date)}</span>
    </div>
  )
}
                                                          مقدار اولیه با تایپ DateObject

                                                          در مثال بالا برای تبدیل String به DateObject ، شرطی در متد onClick دکمه سابمیت قرارداده شده که ممکنه کمی گیج کننده باشه.

                                                          دلیل اونم اینه که مقدار اولیه ای که در state نگهداری میشه از نوع استرینگ هست، برای ساده تر شدن، مثال بالا را میشه به صورت زیر تعییر داد.

                                                          import React, { useState } from "React"
import DatePicker, { DateObject } from "react-multi-date-picker"

export default function example() {
  const [value, setValue] = useState(
    new DateObject({
      calendar: "persian",
      date: "1400/01/01"
    })
  )
  
  const [date, setDate] = useState()
      
  return(
    <div>
      <DatePicker
        calendar="persian"
        locale="fa"
        calendarPosition="auto-right"
        value={value}
        onChange={setValue}
      />
      <button
        onClick={() => setDate(value.toDate())}
      >
        سابمیت
      </button>
      <span>{"" + (date ?? "")}</span>
    </div>
  )
}
                                                          چرا DateObject ؟

                                                          اگر مثال های قبلی رو دیده باشید، ممکنه براتون این سوال پیش بیاد که DateObject دیگه چیه و چرا انقدر تکرار شده 😁

                                                          در جواب باید بهتون بگم که برای یکی از پروژه هام که چند زبانه طراحی شده بود و همینطور با تغییر زبان، تقویم و تاریخش هم به زبان انتخاب شده تغییر میکرد ، مجبور بودم هم از دیت پیکر های مختلف استفاده کنم هم طبیعتا چون هر دیتپیکری برای اجرا به یک کتابخانه خاص متصله ( مثلا یکی میخواد از moment jalali استفاده کنه یکی میخواد از moment hijri استفاده کنه دوباره همه اینا وابسته به خود مومنت هستن) مجبور بودم همه اون کتابخونه ها رو داخل پروژه import کنم.

                                                          تا اینجا مشکل زیادی نبود فقط یکم حجم کار برای import کردن اینهمه کتابخانه بالا میرفت ، مشکل اصلی از اونجا پیدا شد که هر کدام از این دیت پیکر ها ، پیکر بندی های مخصوص به خودشون رو داشتن و کارو برای من سخت میکردن. برای همین تصمیم گرفتم خودم از صفر، یک کتابخانه کار با زمان بنویسم و براش یک کتابخانه دیگه دیتپیکر هم بنویسم تا بتونم با یک پیکربندی واحد از هر چهارتا تقویم و زبان پشتیبانی کنم. اینجوری بود که سر و کله DateObject پیدا شد 🤣

                                                          حقیقتش اینه که در در وهله اول ممکنه کار باهاش یکم عجبب و غریب باشه ولی بعد مشاهده چند تا مثال مختلف که در ادامه میارم میبینید که اونچنانم پیچیده نیست.

                                                          برای مشاهده مثال های بیشتر به صفحه گیت هاب DateObject مراجعه کنید.

                                                          تولید تاریخ میلادی

                                                          همانطور که تا حالا متوجه شدید برای تولید یک نمونه از DateObject باید از کد new DateObject() استفاده کنید. اگر کد بالا رو بدون هیچ کانستراکتوری وارد کنید براتون یک نمونه از DateObject با تقویم میلادی (gregorian) و locale انگلیسی (en) ساخته میشه مثلا برای امروز که به میلادی 2021/01/14 هست اگر من یک new DateObject() تعریف کنم، یک نمونه از همین تاریخ برای من تولید میشه:

                                                          const date = new DateObject()

console.log(date.year) //2021
console.log(date.month.number) //1
console.log(date.day) //14
console.log(date.calendar) //gregorian
console.log(date.locale) //en

                                                          حالا میتونید با متد format تاریخ خودتون رو فرمت کنید :

                                                          console.log(date.format()) //2021/01/14

                                                          همچنین اگر توکن خاصی مد نظرتون هست میتونید داخل متد فرمت اون token رو به صورت پارامتر وارد کنید :

                                                          console.log(date.format("dddd DD MMMM YYYY")) //Thursday 14 January 2021

                                                          و اگر داخل توکن هاتون از مقداری استفاده میکنید که میخواید ignore شه ، اون مقادیر را به صورت آرایه در پارامتر دوم تعریف کنید

                                                          console.log(
  date.format(
    "Date: YYYY/MM/DD Time: HH:mm:ss",
    ["Date", "Time"]
  )
) //Date: 2021/01/14 Time: 19:29:09

                                                          const date = new DateObject({ calendar: "persian" })

console.log(date.format()) //1399/10/25

                                                          تولید تاریخ شمسی با استفاده از متغیر رشته ای

                                                          برای تولید تاریخ شمسی از روی String، یک آبجکت با دو پراپرتی date و calendar را به صورت زیر در constructor قرار بدید.

                                                          فقط به این نکته توجه کنید که تاریخ وارد شده باید از فرمت زیر تبعیت کند در غیر اینصورت به صورت اشتباه تولید میشه:

                                                          YYYY/MM/DD hh:mm:ss.zzz a
                                                          const date = new DateObject({
  date:"1400/01/01",
  calendar:"persian"
})

console.log(date.format()) //1400/01/01
console.log(date.calendar) //persian
console.log(date.year) //1400
console.log(date.weekDay.name) //YekShanbeh
                                                          تولید تاریخ شمسی توسط مقدار عددی

                                                          نکته ای که لازم هست اینجا بگم اینه که من کلمه سال رو داخل ignoreList قرار ندادم ، چون حروفش فارسی هست و خود به خود ignore میشه.

                                                          const date = new DateObject({
  year:1399,
  month:10,
  day:22,
  calendar:"persian"
})

console.log(date.format("dddd DD MMMM سال YYYY")) //دوشنبه ۲۲ دی سال ۱۳۹۹

                                                          const date = new DateObject({
  date: new Date(2020, 1, 15),

console.log(date.format()) //1399/10/26

                                                          خب اگه اینجا برید بهتون ثانیه یونیکس همون لحظه رو میده.

                                                          زمان یونیکس در این لحظه ای که دارم این مطلب رو مینویسم 1610720390 هست.

                                                          فقط توجه کنید که این مقدار به ثانیه هست و ما باید به میلی ثانیه تبدیلش کنیم پس ضرب در 1000 اش میکنیم :

                                                          const date = new DateObject({
  date: 1610720390 * 1000,
  calendar: "persian"

console.log(date.format()) //1399/10/26

                                                          console.log(date.toUnix()) //1610720390

                                                          انتخاب چند زمان

                                                          برای فعال شدن انتخاب چند زمان باید پراپرتی multiple رو برابر با true کنید.

                                                          فقط یادتون باشه که در حالت multiple به جای یک date .خاص ، آرایه ای از زمان ها رو دریافت میکنید، اگر هم میخواید مقادیری رو از قبل تعریف کنید اونهارو داخل آرایه قرار بدید

                                                          من تو این مثال امروز و فردا رو داخل آرایه قرار میدم تا ببینید چجوری میشه بعد تو پروژه خودتون تغییرش بدید یا حذفش کنید.

                                                          import React, { useState } from "react"
import DatePicker from "react-multi-date-picker"
      
export default function Example() {
const [value, setValue] = useState([
  new DateObject({ calendar: "persian" }), //امروز
  new DateObject({ calendar: "persian" }).add(1, "day") //فردا
])

return (
    <DatePicker
    calendar="persian"
    locale="fa"
    calendarPosition="auto-right"
    value={value}
    onChange={setValue}
    multiple
  />
)
      }
                                                          پنل تاریخ ها

                                                          اگر خواستید حالت multiple یا range رو فعال کنید ، میتونید از پلاگین DatePanel هم استفاده کنید تا لیست تاریخ هارو هم داشته باشید.

                                                          برای مشاهده لیست تاریخ ها باید پلاگین DatePanel رو import کنید:

                                                          import React from "react"
import DatePicker from "react-multi-date-picker"
import { DatePanel } from "react-multi-date-picker/plugins"

export default function Example() {
  return (
    <DatePicker
      calendar="persian"
      locale="fa"
      calendarPosition="auto-right"
      multiple
      plugins={[
        <DatePanel position="left" />
      ]}
    />
  )
}
                                                          انتخاب دامنه ای از تاریخ ها

                                                          برای استفاده از حالت range هم باید مقدار value تون آرایه باشه . حالا میتونه این آرایه خالی باشه یعنی [] باشه که در این صورت هیچ دامنه ی تاریخی در دیتپیکرتون نشون داده نمیشه تا خود کاربر یکی رو انتخاب کنه، یا میتونید خودتون یک مقدار رو به عنوان مقدار اولیه تعریف کنید. فقط نکته ای که باید در هنگام مشخص کردن مقدار اولیه در نظر بگیریید اینه که در حالت range آرایه شما باید یک یا دو تا تاریخ داشته باشه و اگه از دوتا بیشتر باشه فقط تاریخ اول و آخر موجود در اون آرایه در نظر گرفته میشن.

                                                          تعریف کردن مقدار اولیه هم که خیلی سادست ، من اینجا یک ارایه که داخلش دوتا تاریخ هست که اولیش 4 روز قبل و دومیش 4 روز بعد هست رو در آرایه قرار میدم تا ببینید چجوری کار میکنه :

                                                          import React from "react"
import DatePicker from "react-multi-date-picker"
import { DatePanel } from "react-multi-date-picker/plugins"

export default function Example() {
  const [value, setValue] = useState([
    new DateObject({ calendar: "persian" }).subtract(4, "days"), //چهار روز قبل
    new DateObject({ calendar: "persian" }).add(4, "days") // چهار روز بعد
  ])

  return (
    <DatePicker
      calendar="persian"
      locale="fa"
      calendarPosition="auto-right"
      value={value}
      onChange={setValue}
      range
      plugins={[
        <DatePanel position="left" />
      ]}
    />
  )
}
                                                          محدود کردن دریافت زمان

                                                          برای محدود کردن دریافت زمان مقدار حداقل رو داخل پراپرتی midDate و مقدار حداکثر رو هم داخل پراپرتی maxDate قرار بدید.

                                                          در اینجا من حداقل رو اول همین ماه و حداکثر رو آخر همین ماه قرار میدم و همونطور که میبینید ماه های بعد و قبل از این ماه غیر فعال میشن.

                                                          توجه کنید که برای رفتن به اول و آخر ماه به ترتیب از متد های toFirstOfMonth و toLastOfMonth استفاده کردم ولی شما اگر با DateObject راحت نیستید میتونید همین کار رو با استرینگ یا دیت هم کنید ، من اینجا رفتن به اول و آخر ماه با استفاده از دیت رو میزارم تا ببینید:

                                                          const date = new Date()

date.setDate(1)

let firstOfMonth = new Date(date) //اول ماه میلادی

date.setMonth(date.getMonth() + 1)
date.setDate(0)

let lastOfMonth = new Date(date) //آخر ماه میلادی

                                                          ولی استفاده ازش رو توصیه نمیکنم چون فقط برای تاریخ میلادی کاربرد داره.

                                                          <DatePicker
  calendar="persian"
  locale="fa"
  calendarPosition="auto-right"
  minDate={new DateObject({ calendar: "persian" }).toFirstOfMonth()} 
  maxDate={new DateObject({ calendar: "persian" }).toLastOfMonth()}
/>
                                                          استفاده از DatePicker در وسط صفحه

                                                          تا اینجا هرچی مثال زدم برای گوشه سمت راست صفحه بهینه بود یعنی باید مقدار calendarPosition رو auto-right قرار میدادید ، حالا تو این مثال و مثال بعدی موقعیت های دیگه رو هم توضیح میدم

                                                          اگر بخواید datepicker تون هر جایی غیر از گوشه ها باشه میتونید اصلا از calendarPosition استفاده نکنید تا خودش calendar رو وسط input بزاره مثلا اینجا datepicker مون وسط صفحه هست و از calendarPosition استفاده نمیکنیم:

                                                          <div style={{ textAlign: "center" }}>
  <DatePicker
    calendar="persian"
    locale="fa"
  />
</div>

                                                          مگرا بخواید به زور calendar رو تو موقعیت ای که خودتون میخواید قرار بدید که در این صورت باید مقدار calendarPosition رو به صورت موقعیت محور x - موقعیت محور y تعریف کنید، مثلا در پایین مقدار محور y رو بالا و مقدار محور x رو چپ قرار میدم.

                                                          <div style={{ textAlign: "center" }}>
  <DatePicker
    calendar="persian"
    locale="fa"
    calendarPosition="top-left"
  />
</div>
                                                          استفاده از DatePicker در گوشه ها

                                                          خب برای گوشه سمت راست که خیلی مثال زدیم فقط میمونه گوشه سمت چپ صفحه.

                                                          اینجا موقعیت تقویم نسبت به input مون auto-left هست ، یعنی در محور y آزاده و ممکنه بنا به وضعیت scroll، بالا یا پایین input باشه، موقعیت محور x هم چپ انتخاب شده.

                                                          برای استایل input هم textAlign رو چپ قرار دادم.

                                                          <div style={{ textAlign: "left" }}>
  <DatePicker
    calendar="persian"
    locale="fa"
    calendarPosition="auto-left"
    style={{ textAlign: "left" }}
  />
</div>
                                                          قرمز کردن روز های جمعه

                                                          برای قرمز کردن روزهای جمعه دو روش وجود داره ، روش اول یکم سخت تره و احتیاج به کد زنی بیشتر داره ولی در عوض یک استیت کمتر نیاز داره که میتونید تو صفحه اصلی پیداش کنید (عبارت Styling Persian Weekends رو جستجو کنید)، روش دوم هم استفاده از پلاگین هست که هوشمند تر هم هست یعنی خودش با توجه به تقویمی که تعریف کردید، روز یا روزهای آخر هفته رو قرمز میکنه:

                                                          import DatePicker from "react-multi-date-picker"
import {Weekends} from "react-multi-date-picker/plugins"

export default function Example() {
  const [value, setValue] = useState()
  const [mapDays, setMapDays] = useState()

  return (
    <DatePicker
            calendar="persian"
            locale="fa"
            calendarPosition="auto-right"
            value={value}
            onChange={setValue}
            mapDays={mapDays}
            plugins={[
              <Weekends setMapDays={setMapDays} />
            ]}
          />
  )
}
                                                          استفاده از پلاگین header

                                                          به عنوان آخرین مثال میخوام استفاده از پلاگین هدر هم نشون بدم ، بقیه مثال ها رو (که شامل همین مثال هم میشه) میتونید از صفحه اصلی یا صفحه پلاگین ها مشاهده کنید.

                                                          import DatePicker from "react-multi-date-picker"
      import {DatePickerHeader} from "react-multi-date-picker/plugins"

      export default function Example() {
        return (
          <DatePicker
            calendar="persian"
            locale="fa"
            calendarPosition="auto-right"
            plugins={[
              <DatePickerHeader position="left" size="medium" />
            ]}
          />
        )
      }
